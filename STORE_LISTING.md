# Chrome Web Store Listing

## Title
Table Exporter

## Short Description (max ~132 chars)
Export any web table to CSV, Excel, JSON, TSV, or Markdown in one click.

## Detailed Description
Table Exporter lets you capture any table on a webpage and export it instantly.

Features:
- Detects tables across iframes and embedded content
- Export formats: CSV, TSV, Excel (XLSX), JSON, Markdown
- Clean preview with table metadata
- Works on most sites with standard HTML tables

Privacy:
- No tracking or analytics
- No data collection or transmission
- Data stays on your machine
